Beyond the New

"Beyond the New" is a web adaptation of the same-titled written manifesto by Hella Jongerius & Louise Schouwenberg. The requirements of this assignment were to create a text-based webpage and utilize CSS and animations to create an interactive experience while navigating the webpage, specifically the interaction of scrolling. The goal of this assignment was to better understand CSS and gain experience with coding animations.

Inspiration

I chose to make a webpage based on the manifesto "Beyond the New" because, in my typography class, we were creating posters using the contents of a manifesto of our choice. I decided that I would make a digital version of my final poster.

Click here to read the full manifesto.

Process

To begin this project, I laid out the text of the manifesto in HTML. I put each bullet point of the manifesto in seperate divs because I had the idea to position the text as "sticky" to create a "stacking" effect as the user scrolls. I then went to Google Fonts to find the right typefaces. Because I used "Baskerville" in my poster, I decided that I would use this for the body. I couldn't find the sans-serif typeface that I used, so I decided to use a similar one called "Heebo." I also went into CSS and began to add color to the text based on the hex codes of my poster. After styling everything I had to start creating the animations

First I decided to work on the title:"Beyond the New, a search for ideals in design." Prior to this assignment, I had never worked with coding animations before. After a bit of experimentation, I found that by applying "animation-delay" to certain animations I could time when each block of text appears on the screen. I used the fade-in animation so the text would appear seamlessly. I also chose to make the text fade out so that the main focus would be on the contents of the manifesto and not just the title.

Next, I had to figure out how to make each point of the manifesto "stack" on top of eachother when the user scrolls. I positioned each of these points as "sticky" and set a margin at the top because I wanted "A search for new ideals in design." to act as a header. I also added a background color, the same as the background of the webpage, behind the text of each of these points and applied a blur to it so that, when stacking on top of eachother, they would seem to fade into eachother seamlessly.

If I were to redo this assignment, I would definitely change the layout. The type looks awkward and cramped in the top left corner and the line spacing is way too large. I realized this at the time but because I did not have as much experience with coding as I do now, I simply did not know how to do that when I was making this webpage.
